Quick Notes

Simple rule Use lowercase for common letters and capitals for special Odia sounds like ଟ, ଠ, ଡ, ଢ, ଣ, ଳ, ଋ, ଙ.
Useful examples RuShi → ଋଷି, Ng → ଙ, ny → ନ୍ୟ, nj → ଞ୍ଜ, MM → ଁ
Nasal marks M → ଂ and MM → ଁ. Examples: kaMM → କଁ, maamuuMM → ମାମୁଁ, kuaaMM → କୁଆଁ
